# Android-Friendly Restaurant Bot Demo

This is a simple static demo designed to be easy to upload from a phone.

## What it does
- Shows a clean booking assistant UI
- Handles a simple reservation conversation
- Answers hours, address, and phone questions
- Deploys easily on Vercel

## What it does not do
- No Anthropic API
- No Google Calendar save
- Demo only

## Deploy on Vercel
Upload this project to GitHub, then import it into Vercel.
Because it is static, Vercel can deploy it without build settings or environment variables.
